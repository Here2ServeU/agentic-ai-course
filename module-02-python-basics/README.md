# Module 2 · Python Basics for Agent Builders

*Learn exactly enough Python to build agents. Nothing extra.*

## Four concepts — that is all you need

- **Variables** store information (the goal, the model name, the step count).
- **Functions** do things with information — **every agent tool is a function**.
- **Lists** hold multiple items in order — your `tools` list is one.
- **Dictionaries** label each item — **every AI API message is a dictionary**.

> Remember: variables store, functions do, lists hold, dictionaries label.
> Every agent you build uses all four — constantly.
